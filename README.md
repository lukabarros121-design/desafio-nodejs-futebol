# ⚽ Simulador de Campeonato de Futebol

## 📝 Descrição do Projeto
Este projeto é uma aplicação de console desenvolvida em Node.js que simula um campeonato de futebol em turno único. O sistema utiliza algoritmos matemáticos baseados em atributos de ataque e defesa de cada equipe para calcular os gols de forma dinâmica, distribuindo a pontuação (vitória, empate ou derrota) e gerando uma tabela classificatória ordenada ao final.
